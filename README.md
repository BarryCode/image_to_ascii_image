# Ascii to ascii image
Convert image to ascii image
```
Version 0.2
```
## Usage
```python
import pyascii

my_img = pyascii.Pyascii("Path-to-image", "Path-to-save-ascii-image")
my_img.default_convert()
my_img.image_show()
my_img.image_save()
```
